================
operun Contact-Form
================

The operun Contact-Form Add-On.


Translations
------------

- Deutsch (de)
- English (en)


Installation
------------

Install operun.contactform by adding it to your buildout::

    [buildout]

    ...

    eggs =
        operun.contactform


and then running ``bin/buildout``
